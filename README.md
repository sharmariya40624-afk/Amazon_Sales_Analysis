**Amazon Sales Analysis using SQL**

**Project Overview**
This project analyses an Amazon E-Commerce sales dataset using SQL to understand sales performance and business trends.

**Tools Used**
1. PostgreSQL
2. SQL
3. Excel

**Dataset**
The dataset contains information about Amazon orders such as order ID, category, shipping city, shipping state, quantity, and revenue amount.

**Analysis Performed**
The following business questions were analysed:
1.Total revenue generated
2.Total number of orders
3.Revenue by product category
4.Revenue by state
5.Revenue by city

**Project Structure**
Dataset - original dataset
Queries - SQL queries used for analysis
Results – exported results from SQL queries

**Conclusion**
This analysis demonstrates how SQL can be used to explore E-Commerce sales data and extract useful business insights.
